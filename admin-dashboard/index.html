Lucene

This Lucene Plugin allows for the use of lucene indexing and queries within your app. The current implementation supports use of the _fti endpoint within plugins though corresponding database indexing functions. 

            
            {
                "_id":"_design/foo",
                "fulltext": {
                    "by_subject": {
                        "index":"function(doc) { var ret=new Document(); ret.add(doc.subject); return ret }"
                    },
                    "by_content": {
                        "index":"function(doc) { var ret=new Document(); ret.add(doc.content); return ret }"
                    }
                }
            }
            
        
            
            http://host/database/_fti/_design/foo/by_subject?q=hello
            http://host/database/_fti/_design/foo/by_content?q=hello
